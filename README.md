# Job Application Tracker

A simple and user-friendly web application to track and manage job applications efficiently.

## Overview
This project helps users organize their job search by allowing them to add, view, and track the status of job applications in one place.

## Features
- Add new job applications
- Track application status (Applied, Interview, Offer)
- Clean and easy-to-use interface
- Responsive design for different screen sizes

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
Users can enter the company name, job role, and application status. The application then displays the job entries dynamically on the page.

## Project Structure
- `index.html` – Main structure of the application
- `style.css` – Styling and layout
- `script.js` – Functionality and logic

## Future Improvements
- Add delete and edit functionality
- Store data using local storage
- Improve UI with advanced styling
